workspace issues:
Tailwind CSS IntelliSense may not always work. to fix this.

ctrl + p ==> then type ">settings.json", enter "open workspace settings (JSON)"

{
    "tailwindCSS.includeLanguages": {
    "html": "html",
    "javascript": "javascript",
    "css": "css"
},
"editor.quickSuggestions": {
    "strings": true
}
}
